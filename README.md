#### simple groovy application with gradle

This is a simple template for groovy application with gradle tool.

#### compile、build、run

```bash
$./gradlew run
```
```bash
$./gradlew test
```
```bash
$./gradlew groovydoc
```
```bash
$./gradlew assembleDist
```
